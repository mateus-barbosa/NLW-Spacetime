## Projeto

Esse projeto é uma capsula do tempo desenvolvida para exibir memórias em uma <i>time-line</i>

## Tecnologias

Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e GitHub
